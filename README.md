# Juego del Ahorcado

Implementación digital del clásico juego de papel "El Ahorcado" en Python.

## Versiones disponibles

### 🎨 Versión con Interfaz Gráfica (Recomendada)
Interfaz linda con botones interactivos y diseño moderno.

```bash
python ahorcado_gui.py
```

### 💻 Versión de Consola
Versión clásica en terminal.

```bash
python ahorcado.py
```

## Cómo jugar

1. El computador selecciona una palabra al azar
2. Debes adivinar la palabra letra por letra
3. Cada letra correcta se revela en su posición
4. Cada letra incorrecta dibuja una parte del ahorcado
5. Tienes 6 intentos fallidos antes de perder
6. Ganas si adivinas la palabra completa antes de completar el dibujo

## Características de la versión GUI

- ✨ Interfaz gráfica moderna y colorida
- 🎨 Diseño atractivo con colores profesionales
- 🖱️ Botones interactivos para cada letra del abecedario
- 🎭 Dibujo animado del ahorcado
- 🔄 Botón para iniciar nuevo juego
